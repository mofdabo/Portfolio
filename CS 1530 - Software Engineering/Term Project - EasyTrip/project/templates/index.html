Welcome to EasyTrip



The EasyTrip software takes in locations the user wishes to visit and plots the order they should be visited in based on factors such as closing times, busy times, reservations, tickets, etc.


The software is to be used by those planning trips to different cities or other locations who hope to maximize the time they spend at each attraction. EasyTrip makes travel more efficient, increases the benefit the user receives from each location they visit, and ensures that the user can visit as many of their desired destinations as possible promptly.
